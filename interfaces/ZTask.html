    • Defined in run-z/run-z/src/core/tasks/task.ts:62

    Represents this task as a prerequisite of another one.

    By default a grouping task treats the first argument as a sub-task name and the rest of arguments as arguments to this sub-task. A task of any other type returns a call to itself.
