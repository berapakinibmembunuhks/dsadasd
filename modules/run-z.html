Const ZDepsFirstBatches

ZDepsFirstBatches: ZBatchRule<ZDepsFirstBatches> = ZDepsFirstBatches$
  • Defined in run-z/run-z/src/core/batches/deps-first-batches.rule.ts:118

Dependencies-first batched tasks execution rule.

When enabled, any task targeted dependent package is executed before the tasks targeted its dependencies. It also allows to run tasks from independent packages in parallel to each other.

Dependencies-first mode is enabled by default.

batchNamed

  • batchNamed(this: void, planner: ZBatchPlanner): Promise<void>
    • Defined in run-z/run-z/src/core/batches/batcher.ts:87

    Batches tasks in all named batches.

    A named batch is defined by (preferably grouping) task with the name like one of:

    • batch-name/task-name. Such batch is processed when the batched task name matches task-name.

    • batch-name/*. Such batch is processed, unless there is another named batch batch-name/task-name defined with the batched task name matching task-name.

    If target package has no matching named batches then batches the task by default batcher.

    Parameters

    • this: void
    • planner: ZBatchPlanner

      Batch execution planner to record batched task calls to.

    Returns Promise<void>

    A promise resolved when batch execution planned.
